Privacidad y cookies

sandramangas.com es una web educativa y divulgativa, sin actividad económica, publicidad ni formularios. No puedes registrarte ni suscribirte y no vendemos tus datos.

Responsable y contacto

La responsable del sitio es Sandra Mangas. Para cualquier consulta o para ejercer tus derechos de protección de datos puedes escribir a sandeeacocina@gmail.com.

Datos que puedes enviar por correo

La web no recoge datos mediante formularios. Si decides escribir por correo electrónico, trataremos la dirección y la información que incluyas únicamente para atender y responder tu consulta. La base jurídica es tu propia solicitud y el envío voluntario del mensaje. Los datos se conservarán durante el tiempo necesario para responder y, después, solo durante los plazos necesarios para atender posibles responsabilidades legales.
